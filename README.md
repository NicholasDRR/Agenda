# AGENDA

## Descrição
Agenda feita em Django

## Como Usar

### Instalando
Clone este repositorio:

```git clone https://github.com/NicholasDRR/Agenda.git```

Entre no repositorio:
```cd Agenda```

Instalar Requisitos:
```pip install -r REQUIREMENTS.txt```
